# 📚 Multi-Cluster Kubernetes Reference Links

Here is a curated list of official documentation, GitHub repositories, and specifications to help you deepen your understanding of global operations:

---

## 🌐 Kubernetes Official SIGs
*   [Kubernetes SIG Multicluster](https://github.com/kubernetes/community/tree/master/sig-multicluster) — The official special interest group managing multi-cluster specifications and APIs.
*   [Multi-Cluster Services (MCS) API Spec](https://github.com/kubernetes-sigs/mcs-api) — KEP-1645 standardizing service discovery across clusters.

---

## 🤝 Federation Frameworks
*   [Karmada Official Website](https://karmada.io/) — Documentation for the multi-cluster orchestration engine.
*   [Open Cluster Management (OCM)](https://open-cluster-management.io/) — Guide to governance, placement, and registration models.

---

## 🔌 Cross-Cluster Networking
*   [Cilium ClusterMesh Guide](https://docs.cilium.io/en/stable/network/clustermesh/clustermesh/) — Step-by-step setup for eBPF-powered routing meshes.
*   [Submariner Documentation](https://submariner.io/) — Tunnels and Globalnet NAT configurations.

---

## 🗺️ Global Traffic Steering
*   [K8gb (Kubernetes Global Load Balancer)](https://www.k8gb.io/) — Lightweight Cloud Native GSLB controller for Kubernetes.
*   [CoreDNS GeoIP Plugin](https://github.com/coredns/geoip) — Resolving queries based on GeoIP databases.
